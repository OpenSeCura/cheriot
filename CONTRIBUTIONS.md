# Contributing to Cherified

Want to contribute? Great! First, a few objectives:

## Contributor License Agreement

Contributions to this project must be accompanied by a Contributor License
Agreement. You (or your employer) must keep the agreement on file and sign it
before we can accept your pull requests. To make this as easy as possible, we
use a CLA bot.

When you make a pull request, the CLA bot will check if you have signed the
Google CLA. If you haven't, it will provide a link where you can sign it
online. You only need to do this once for all Google projects.

*   If you are an individual writing original source code and you're sure you
    own the intellectual property, then you'll need to sign an [individual CLA](https://cla.developers.google.com/about/google-individual).
*   If you work for a company, they must sign a [corporate CLA](https://cla.developers.google.com/about/google-corporate) to
    allow you to contribute.

## Submission Process

1. **Fork** the repository and create your branch from `main`.
2. **Implement** your changes.
3. Ensure your commits have descriptive messages and all source files have the correct **license header**.
4. **Submit a Pull Request** to the `main` branch.

## Community Guidelines

This project follows [Google's Open Source Community Guidelines](https://opensource.google/conduct/).
